## Features

**Translation**
- 6 translation engines: Google, Bing, Yandex (free) + DeepL, Gemini, ChatGPT (API key)
- Auto language detection, real-time translation, alternative suggestions
- Automatic engine fallback on failure
- Element select mode — click any part of a page to translate in-place
- Side panel for continuous translation
- Full page and link translation

**RTL Support**
- Full right-to-left rendering for Persian, Arabic, Hebrew, Urdu
- Automatic bidirectional text fixing (3 levels: gentle, medium, aggressive)
- Smart threshold — only activates when RTL character percentage exceeds your setting
- Persian font injection (Vazirmatn or custom)
- Special Instagram & Twitter/X support

**Instagram Downloader**
- Download photos, videos, stories, reels, and profile pictures
- Bulk download all stories at once
- Dual-engine architecture: Tondar (React Fiber) + Borna (DOM-based) with automatic fallback
- Smart retry with exponential backoff
- 8 granular toggles for precise control

**Security & Privacy**
- API keys encrypted with AES-256-GCM
- Zero data sent to external servers
- No analytics, no tracking, no ads
- Everything stays local on your device

**Productivity**
- Keyboard shortcuts (Ctrl+Space, Alt+Shift+S, and more)
- Translation history (up to 500 entries with search)
- Text-to-speech
- Import/export settings (encrypted)
- Dark, light, and system themes
