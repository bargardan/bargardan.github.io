Installation Guide

Install and set up Bargardan in a few simple steps

1

Open Chrome Web Store

Click the "Install from Chrome Web Store" button or search for "Bargardan" in the Chrome Web Store.

Chrome Web Store
2

Add the Extension

Click the "Add to Chrome" button.

3

Confirm Permissions

Click "Add Extension" in the popup dialog.

4

Pin to Toolbar

Click the Chrome puzzle icon and pin Bargardan so it's always accessible.

5

First Translation

Select some text and press Ctrl+Space. Congratulations — Bargardan is ready!

6

Customize Settings

Right-click the extension icon > Settings. Configure your translation engine, theme, and other options.

7

Set Keyboard Shortcuts

Go to chrome://extensions/shortcuts and configure your preferred shortcuts.
