Summary

Bargardan does not send any data to external servers. All information is stored and processed locally on your device.

Information We Collect

Bargardan only stores the following data locally in your browser: • Extension settings: Source and target language, selected translation engine, appearance preferences. • Translation history: Previously translated texts for quick access (can be deleted by the user at any time). • API keys: If you use DeepL, ChatGPT (OpenAI), or Google Gemini, your API key is stored encrypted (AES-256-GCM) and only locally on your device. These keys are never sent to any external server and are only used when sending a translation request to the respective service. All data is stored using chrome.storage.local and is never sent to any external server.

Third-Party Translation Services

When you translate text, Bargardan sends the selected text to the translation engine you have chosen: • Google Translate • Microsoft Bing Translator • DeepL • Yandex Translate • ChatGPT (OpenAI) • Google Gemini Each of these services has its own independent privacy policy. Bargardan only sends the selected text for translation and does not add any other identifying information.

Instagram Download Feature

Bargardan provides the ability to download publicly available Instagram photos and videos. During this process: • Access to instagram.com and cdninstagram.com is used solely for downloading content. • No account information is stored or transmitted. • Downloaded files are saved directly to your device. • No Instagram data is sent to our servers or any third party.

Extension Permissions

Explanation of each permission: • storage: Store settings and translation history locally. • activeTab: Access selected text on the current page for translation. • contextMenus: Add translation options to the right-click menu. • scripting: Inject translation and download scripts into web pages. • downloads: Save Instagram downloads to your device. • sidePanel: Display the translation side panel. • tts: Text-to-speech for translated text.

Data Sharing

Bargardan does not share, sell, or transfer any data to third parties — except for the translation text sent to your chosen translation service.

Analytics and Tracking

Bargardan does not use any analytics, tracking, or advertising tools.

API Key Security

API keys for DeepL, ChatGPT, and Gemini engines are encrypted with AES-256-GCM before storage. These keys are never sent to any external server and are only used when sending a translation request to the respective service.

Local Storage

All data is stored on your device via chrome.storage.local. No data is sent to external servers.

Translation Data

Text you translate is only sent to your chosen translation engine. Bargardan does not keep any copy of these texts and does not add any other identifying information.

User Rights

You can at any time: • Delete your translation history from the extension settings. • Remove all stored data by uninstalling the extension. • Manage extension permissions from your browser settings.

Changes to This Privacy Policy

If this policy changes, the updated version will be published on this page and the last updated date will be changed.

Contact

If you have any questions about privacy, please contact us at itsrezajax@gmail.com or through the extension's support page on the Chrome Web Store.
