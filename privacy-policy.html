Privacy Policy

Bargardan Chrome Extension | Last updated: March 2026

Summary: Bargardan does not send any data to external servers. All information is stored and processed locally on your device.

1. Information We Collect

Bargardan only stores the following data locally in your browser:

  • Extension settings: Source and target language, selected translation engine, appearance preferences.
  • Translation history: Previously translated texts for quick access (can be deleted by the user at any time).
  • API keys: If you use DeepL, ChatGPT (OpenAI), or Google Gemini, your API key is stored encrypted (AES-256-GCM) and only locally on your device. These keys are never sent to any external server and are only used when sending a translation request to the respective service.

All data is stored using chrome.storage.local and is never sent to any external server.

2. Third-Party Translation Services

When you translate text, Bargardan sends the selected text to the translation engine you have chosen. These services include:

  • Google Translate
  • Microsoft Bing Translator
  • DeepL
  • Yandex Translate
  • ChatGPT (OpenAI)
  • Google Gemini

Each of these services has its own independent privacy policy. Bargardan only sends the selected text for translation and does not add any other identifying information.

3. Instagram Download Feature

Bargardan provides the ability to download publicly available Instagram photos and videos. During this process:

  • Access to instagram.com and cdninstagram.com is used solely for downloading content.
  • No account information is stored or transmitted.
  • Downloaded files are saved directly to your device.
  • No Instagram data is sent to our servers or any third party.

4. Extension Permissions

Explanation of each permission:

  • storage: Store settings and translation history locally.
  • activeTab: Access selected text on the current page for translation.
  • contextMenus: Add translation options to the right-click menu.
  • scripting: Inject translation and download scripts into web pages.
  • downloads: Save Instagram downloads to your device.
  • sidePanel: Display the translation side panel.
  • tts: Text-to-speech for translated text.

5. Data Sharing

Bargardan does not share, sell, or transfer any data to third parties — except for the translation text sent to your chosen translation service.

6. Analytics and Tracking

Bargardan does not use any analytics, tracking, or advertising tools.

7. User Rights

You can at any time:

  • Delete your translation history from the extension settings.
  • Remove all stored data by uninstalling the extension.
  • Manage extension permissions from your browser settings.

8. Changes to This Privacy Policy

If this policy changes, the updated version will be published on this page and the last updated date will be changed.

9. Contact

If you have any questions about privacy, please contact us through the extension's support page on the Chrome Web Store.

سیاست حفظ حریم خصوصی

خلاصه: برگردان هیچ داده‌ای را به سرور خارجی ارسال نمی‌کند. تمام اطلاعات به‌صورت محلی (روی دستگاه شما) ذخیره و پردازش می‌شوند.

۱. اطلاعاتی که جمع‌آوری می‌شود

برگردان فقط اطلاعات زیر را به‌صورت محلی روی مرورگر شما ذخیره می‌کند:

  • تنظیمات افزونه: زبان مبدأ و مقصد، موتور ترجمه انتخابی، تنظیمات ظاهری
  • تاریخچه ترجمه: متن‌های ترجمه‌شده برای دسترسی سریع (قابل حذف توسط کاربر)
  • کلیدهای API: در صورت استفاده از سرویس‌های DeepL، ChatGPT (OpenAI) یا Google Gemini، کلید API شما به‌صورت رمزنگاری‌شده (AES-256-GCM) و فقط به‌صورت محلی روی دستگاه شما ذخیره می‌شود. این کلیدها هرگز به سرور خارجی ارسال نمی‌شوند و فقط هنگام ارسال درخواست ترجمه به سرویس مربوطه استفاده می‌شوند.

این داده‌ها با استفاده از chrome.storage.local ذخیره می‌شوند و هرگز به سرور خارجی ارسال نمی‌شوند.

۲. سرویس‌های ترجمه (طرف سوم)

هنگام ترجمه متن، برگردان متن انتخاب‌شده را به موتور ترجمه‌ای که خودتان انتخاب کرده‌اید ارسال می‌کند. این سرویس‌ها عبارتند از:

  • Google Translate
  • Microsoft Bing Translator
  • DeepL
  • Yandex Translate
  • ChatGPT (OpenAI)
  • Google Gemini

هر کدام از این سرویس‌ها سیاست حریم خصوصی مستقل خود را دارند. برگردان فقط متن انتخاب‌شده را برای ترجمه ارسال می‌کند و هیچ اطلاعات شناسایی دیگری اضافه نمی‌کند.

۳. قابلیت دانلود اینستاگرام

برگردان امکان دانلود تصاویر و ویدیوهای عمومی اینستاگرام را فراهم می‌کند. در این فرآیند:

  • دسترسی به instagram.com و cdninstagram.com فقط برای دانلود محتوا استفاده می‌شود
  • هیچ اطلاعات حساب کاربری شما ذخیره یا ارسال نمی‌شود
  • فایل‌های دانلودشده مستقیماً روی دستگاه شما ذخیره می‌شوند
  • هیچ داده‌ای از اینستاگرام به سرورهای ما یا طرف سوم ارسال نمی‌شود

۴. مجوزهای افزونه

توضیح دلیل هر مجوز:

  • storage: ذخیره تنظیمات و تاریخچه ترجمه به‌صورت محلی
  • activeTab: دسترسی به متن انتخاب‌شده در صفحه فعلی برای ترجمه
  • contextMenus: افزودن گزینه ترجمه به منوی راست‌کلیک
  • scripting: تزریق اسکریپت ترجمه و دانلود در صفحات وب
  • downloads: ذخیره فایل‌های دانلودی اینستاگرام روی دستگاه
  • sidePanel: نمایش پنل جانبی ترجمه
  • tts: تلفظ صوتی متن ترجمه‌شده

۵. اشتراک‌گذاری داده

برگردان هیچ داده‌ای را با طرف سوم به اشتراک نمی‌گذارد، نمی‌فروشد و منتقل نمی‌کند — به‌جز متن ترجمه که به سرویس ترجمه انتخابی شما ارسال می‌شود.

۶. تحلیل و ردیابی

برگردان از هیچ ابزار تحلیلی (analytics)، ردیابی (tracking) یا تبلیغاتی استفاده نمی‌کند.

۷. حقوق کاربر

شما می‌توانید در هر زمان:

  • تاریخچه ترجمه خود را از تنظیمات افزونه حذف کنید
  • تمام داده‌های ذخیره‌شده را با حذف افزونه پاک کنید
  • مجوزهای افزونه را از تنظیمات مرورگر مدیریت کنید

۸. تغییرات در سیاست حریم خصوصی

در صورت تغییر این سیاست، نسخه به‌روزشده در همین صفحه منتشر خواهد شد و تاریخ آخرین به‌روزرسانی تغییر خواهد کرد.

۹. تماس

در صورت داشتن سؤال درباره حریم خصوصی، از طریق صفحه پشتیبانی افزونه در Chrome Web Store با ما تماس بگیرید.
